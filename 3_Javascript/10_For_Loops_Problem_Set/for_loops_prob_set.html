For Loops Problem Set

  1. Print all numbers between -10 and 19
  2. Print all numbers between 10 and 40
  3. Print all numbers between 300 and 333
  4. Print all numbers divisible between 5 AND 3 between 5 and 50

Notice that this is the same problem set you did with while loops,
only now you are being asked to do them with for loops.
